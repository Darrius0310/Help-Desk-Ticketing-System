<h1>Help Desk Ticketing System</h1>

<h2>Description</h2>
This section demonstrates utilizing Jira Service Management to simulate real IT help desk operations by creating, prioritizing, and assigning tickets while managing workflow from initiation to resolution. Applied structured triaging, maintained clear work notes, and delivered thorough resolutions, demonstrating the ability to handle support requests efficiently in a professional IT environment.
